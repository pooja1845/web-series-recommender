🎬 Web Series Recommender System

A content-based recommendation system that suggests similar web series based on user selection. Built using Python, Pandas, and Streamlit, and deployed as an interactive web application.

🚀 Live Demo

👉 https://web-series-recommender-9mrpl2unuaa47hzutjkqpm.streamlit.app/

 Features :
     Recommend top 5 similar web series
     Fetch posters, ratings, genre, and year using OMDb API
     Fast and interactive UI with Streamlit
     Handles large datasets using external storage (Google Drive)
     Clean and responsive layout (Selected vs Recommended)
     
Tech Stack :
      Frontend: Streamlit
      Backend: Python
      Libraries: Pandas, Requests, Pickle
      API: OMDb API
      Deployment: Streamlit Cloud / Heroku
      Storage: Google Drive (for large .pkl files)
